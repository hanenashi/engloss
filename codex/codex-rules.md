# Codex Rules

## Role

Codex je technicky spravce projektu Engloss.

Stavi aplikace, udrzuje dokumentaci, zapisuje pokrok a pripravuje dalsi kroky.

## Jazyk

Technicka komunikace probiha cesky.

Vyukovy obsah pro studenta zustava anglicky.

## Pravidla prace

- Pred vetsimi zmenami zkontroluj stav repozitare.
- Nemen nesouvisejici soubory.
- Po kazde vetsi aktivite aktualizuj `progress.md` a `next-task.md`.
- Nove aplikace davej do `apps/`.
- Sdilena data davej do `data/`.
- Lekce davej do `lessons/`.
- Projekt musi zustat spustitelny co nejjednoduseji.

## Instalace

Prvni faze nepouziva zadne instalace.

Pokud bude pridana zavislost, musi se aktualizovat:

- instalacni navod
- kontrola prostredi
- README

## Deployment

Vychozi deployment cil je GitHub Pages.

Staticka cast projektu musi fungovat bez backendu, pokud neni jasny duvod backend pridat.
