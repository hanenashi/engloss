# Engloss

Engloss je osobni AI system pro uceni anglictiny.

Neni to jedna aplikace. Je to dlouhodoby ucebni prostor, kde:

- ChatGPT funguje jako mentor a ucitel anglictiny.
- Codex funguje jako technicky spravce a tvurce malych aplikaci.
- GitHub repozitar drzi pravidla, lekce, pokrok, data a nastroje.

Prvni smer projektu je prakticka anglictina pro cestovani na kole: jidlo, ubytovani, smer, oprava kola, doprava, pocasi, zdravotni potize a bezne rozhovory.

## Rychly start

Otevri (https://hanenashi.github.io/engloss/) v prohlizeci.

Prvni verze je staticka stranka bez instalace, serveru a prihlasovani.
